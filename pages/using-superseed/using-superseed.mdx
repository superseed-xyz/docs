# Using Superseed

## MetaMask

Addresses in your MetaMask will work and be the same on both Ethereum and the Superseed blockchain. However, to submit transactions to the Superseed blockchain you must add the network to your wallet.

### Step-by-step guide

To add Superseed as a custom network in MetaMask:

<Steps>

### Open the MetaMask browser extension.

### Click the network selection dropdown

Open the network selection dropdown menu by clicking the dropdown button at the top-left corner of the extension.

### Click the _Add network_ button.

### Click _Add a network manually_.
