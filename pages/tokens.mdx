# Superseed Token List

## Testnet

ℹ️ **Guidelines**
To list your token on the Superseed testnet, use this form and ensure your token follows the guidelines below:

- Token must be an ERC-20 token
- Token must be deployed on the Ethereum Sepolia testnet
- Token must have a verified contract on Sepolia Ethereum Etherscan
- Token must have a verified contract on Sepolia Superseed Explorer
- Request must be created by a representative of the project or include a verifying tweet from the project Twitter account.
